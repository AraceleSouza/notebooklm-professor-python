# 📚 Referências Utilizadas no Projeto

Este documento reúne as principais fontes utilizadas no NotebookLM durante o desenvolvimento do projeto **“NotebookLM como Professor Particular de Python”**.

As referências foram selecionadas com o objetivo de construir um material de estudo confiável, didático e adequado para iniciantes em Python.

---

# 📺 Videoaulas e Cursos

## 🎥 Curso em Vídeo — Python

Professor: Gustavo Guanabara

Conteúdos utilizados:

* estruturas condicionais;
* estruturas de repetição;
* listas e tuplas;
* funções;
* modularização;
* Programação Orientada a Objetos (POO).

Link:
https://www.cursoemvideo.com/

---

## 🎥 Aulas de Programação Orientada a Objetos

Conteúdos utilizados:

* classes;
* objetos;
* atributos;
* métodos;
* encapsulamento.

Fonte utilizada dentro do NotebookLM para aprofundamento em POO.

# 🧠 Critérios de Seleção das Fontes

As fontes foram escolhidas considerando:

A combinação entre:
