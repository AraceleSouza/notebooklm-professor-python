# 🛠️ Prompts Utilizados no Projeto

Este documento reúne os principais prompts utilizados durante os estudos no NotebookLM para explorar o aprendizado de Python com apoio da Inteligência Artificial.

Além das perguntas realizadas, também foram registrados os objetivos, resultados obtidos e aprendizados adquiridos durante o processo.

---

# 📘 Explicações Didáticas

## 📌 Prompt 1 — Funções em Python

```txt
Explique funções em Python como um professor para iniciantes.
```

### 🎯 Objetivo

Compreender o conceito de funções utilizando linguagem simples e exemplos práticos.

### ✅ Resultado Obtido

O NotebookLM gerou uma explicação passo a passo sobre:

* definição de funções;
* reutilização de código;
* parâmetros;
* retorno;
* exemplos práticos.

### 🧠 Aprendizado

Percebi que prompts mais específicos geram respostas mais organizadas e didáticas.

---

## 📌 Prompt 2 — Listas e Tuplas

```txt
Explique a diferença entre listas e tuplas em Python de forma simples para iniciantes.
```

### 🎯 Objetivo

Entender o conceito de mutabilidade em Python.

### ✅ Resultado Obtido

A IA explicou:

* diferenças entre listas e tuplas;
* uso de colchetes e parênteses;
* quando utilizar cada estrutura;
* exemplos cotidianos para facilitar o entendimento.

### 🧠 Aprendizado

Solicitar explicações “para iniciantes” melhora muito a clareza das respostas.

# ⚠️ Troubleshooting e Cicatrizes

## 🔸 Problema: Respostas muito genéricas

### ✔️ Solução

Adicionar:

* contexto;
* nível de dificuldade;
* objetivo esperado;
* exemplos desejados.

---

## 🔸 Problema: Explicações avançadas demais

### ✔️ Solução

Solicitar respostas:

* “em linguagem simples”;
* “para iniciantes”;
* “com analogias”.

---

## 🔸 Problema: Falta de exemplos práticos

### ✔️ Solução

Solicitar:

* exercícios;
* aplicações reais;
* exemplos comentados.
